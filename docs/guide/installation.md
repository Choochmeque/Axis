# Installation

## Requirements

- **Git** must be installed on your system
- Supported platforms: macOS, Windows, Linux

## Download

Download the latest release from [GitHub Releases](https://github.com/Choochmeque/Axis/releases).

| Platform | Format |
|----------|--------|
| macOS | `.dmg` |
| Windows | `.exe` (NSIS installer) |
| Linux | `.AppImage` |

## macOS

1. Download the `.dmg` file
2. Open the disk image
3. Drag Axis to your Applications folder
4. Launch Axis from Applications

::: tip
On first launch, you may need to right-click and select "Open" to bypass Gatekeeper.
:::

## Windows

1. Download the `.exe` installer
2. Run the installer
3. Follow the installation wizard
4. Launch Axis from the Start menu

## Linux

1. Download the `.AppImage` file
2. Make it executable: `chmod +x Axis-*.AppImage`
3. Run the AppImage

::: tip
You can use [AppImageLauncher](https://github.com/TheAssassin/AppImageLauncher) for better desktop integration.
:::

## Nightly Builds

Want the latest features? Download nightly builds from the [Releases](https://github.com/Choochmeque/Axis/releases) page (tagged with `-nightly`).
