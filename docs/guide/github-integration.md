# GitHub Integration

Connect Axis to GitHub for enhanced workflow.

## Authentication

1. Go to **Settings > GitHub**
2. Click **Sign in with GitHub**
3. Authorize Axis in your browser

## Pull Requests

### Viewing PRs
- See all open PRs for your repository
- Filter by author, reviewer, or label
- View PR details and comments

### Creating a PR
1. Push your branch
2. Click **Create Pull Request**
3. Fill in title and description
4. Select reviewers and labels
5. Click **Create**

### Reviewing PRs
- View changed files
- Leave comments
- Approve or request changes

## Issues

### Viewing Issues
- Browse open and closed issues
- Filter by label, assignee, or milestone

### Creating Issues
1. Click **New Issue**
2. Enter title and description
3. Add labels and assignees
4. Click **Create**

## CI Status

- See build status for commits and PRs
- Click to view workflow details
- Re-run failed workflows

## Notifications

- Get notified of PR reviews
- See mentions and assignments
- Configure in **Settings > Notifications**
