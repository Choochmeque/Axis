# Keyboard Shortcuts

Speed up your workflow with keyboard shortcuts.

## Global

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| Open repository | `Cmd+O` | `Ctrl+O` |
| Settings | `Cmd+,` | `Ctrl+,` |
| Search | `Cmd+P` | `Ctrl+P` |
| Refresh | `Cmd+R` | `Ctrl+R` |

## Repository

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| Commit | `Cmd+Enter` | `Ctrl+Enter` |
| Stage all | `Cmd+Shift+S` | `Ctrl+Shift+S` |
| Unstage all | `Cmd+Shift+U` | `Ctrl+Shift+U` |
| Push | `Cmd+Shift+P` | `Ctrl+Shift+P` |
| Pull | `Cmd+Shift+L` | `Ctrl+Shift+L` |
| Fetch | `Cmd+Shift+F` | `Ctrl+Shift+F` |

## Navigation

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| History view | `Cmd+1` | `Ctrl+1` |
| Staging view | `Cmd+2` | `Ctrl+2` |
| Branches view | `Cmd+3` | `Ctrl+3` |
| Stashes view | `Cmd+4` | `Ctrl+4` |

## Custom Shortcuts

You can customize shortcuts in **Settings > Keyboard Shortcuts**.
