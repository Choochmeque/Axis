# Settings

Configure Axis to your preferences.

## General

- **Language** - Choose UI language
- **Theme** - Light, dark, or system
- **Font Size** - Adjust text size

## Git

- **Default Branch** - Name for new repositories
- **Signing** - GPG/SSH commit signing
- **Merge Strategy** - Default merge behavior

## Editor

- **External Editor** - Set your preferred code editor
- **Diff Tool** - External diff application
- **Merge Tool** - External merge tool

## GitHub

- **Account** - Sign in/out of GitHub
- **Notifications** - Configure alerts

## AI

- **Provider** - Claude, OpenAI, or Ollama
- **API Key** - Your provider credentials
- **Model** - Choose AI model

## Keyboard Shortcuts

- View all shortcuts
- Customize key bindings
- Reset to defaults

## Advanced

- **Proxy** - Network proxy settings
- **Logs** - View application logs
- **Cache** - Clear cached data
