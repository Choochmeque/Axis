# First Repository

## Clone a Repository

1. Click **Clone Repository** on the welcome screen
2. Enter the repository URL (HTTPS or SSH)
3. Choose a local folder
4. Click **Clone**

## Open an Existing Repository

1. Click **Open Repository**
2. Navigate to your local Git repository
3. Select the folder containing `.git`
4. Click **Open**

## Initialize a New Repository

1. Click **Init Repository**
2. Choose an empty folder or existing project
3. Click **Initialize**

Your repository is now ready to use in Axis.
