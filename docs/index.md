## Why Axis?

Axis combines the power of a full-featured Git client with a modern, intuitive interface. Whether you're managing complex branching strategies or just committing your daily work, Axis makes Git accessible without sacrificing power.

### Built for Developers

- **Interactive Rebase** - Visualize and edit your commit history with ease
- **Conflict Resolution** - Intuitive merge conflict resolver with side-by-side comparison
- **Blame View** - Understand code history with syntax-highlighted blame
- **Content Search** - Search across your entire repository history
- **Custom Actions** - Create keyboard shortcuts for your frequent operations

### Enterprise Ready

- **SSH & GPG Signing** - Full support for commit signing
- **Git LFS** - Large file storage support built-in
- **Worktrees** - Manage multiple working directories
- **Submodules** - First-class submodule support
- **GitFlow** - Built-in GitFlow workflow
